Place your .mp3 sound files here.

Each sound button in index.ejs has a data-sound attribute that maps to a filename in this folder.
Example: data-sound="airhorn.mp3" → public/sounds/airhorn.mp3

Default sound buttons and their expected filenames:
 Memes:
  - airhorn.mp3
  - bruh.mp3
  - oof.mp3
  - vine_boom.mp3
  - sad_trombone.mp3
  - emotional_damage.mp3

 Alerts:
  - alert.mp3
  - ding.mp3
  - error.mp3
  - success.mp3

 Music:
  - dramatic.mp3
  - suspense.mp3
  - victory.mp3
  - fail.mp3

 FX:
  - explosion.mp3
  - laser.mp3
  - punch.mp3
  - thunder.mp3

To add new sounds, drop the file here and add a button in views/index.ejs.
